# neural-analysis-suite
A collection of MATLAB-based tools for analyzing and visualizing the neural data collected in my PhD thesis work: Vibrotactile Feature Representation in Distinct Neuronal Subtypes of Primary Somatosensory Cortex

## Working with this repo
1. Acquire data tables necessary for these analysis tools. Reach out to me <mduhain@gmail.com> for copies of the data (too large for GitHub)
2. Start with: 'focusedAnalysisScript.m' and proceed through the first two code sections (loading an active dataset, and some initial analysis).
3. Once you reach the branch point, you can then proceed to any script with the FAS_* prefix to continue.
